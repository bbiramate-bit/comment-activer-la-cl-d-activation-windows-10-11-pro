# comment-activer-la-clé-d-activation-windows-10-11-pro
si votre windows 10/11 vous demande vune clé d'activation, vous pouvez l'activer grâce à l'interface de ligne de commande. suivez juste les intsructions de ce repository

# -- Note: This method requires you to have a Microsoft account connected! This is the method most people may use as this one activates it PERMANENT.--


1. Appuyez sur windows + R to open the Run menu.
2. Entrez 'powershell' dans la CLI.
3. Faite Ctrl + Shift + Enter.
4. dans le PowerShell window, entrez la commande suivante puis Enter: irm https://get.activated.win | iex 
5. Attendez jus'a ce que vous voyez le menu d'activation windows apparaitre
6. choisissez l'option for HWID Activation en tapant au numero associé .
7. attendez que ça soit complet. It might give an SKU value error, but it will complete successfully. If it doesn't try again a few times and it should work.

merci d'avoir suivi ce tuto ! et maintenant utiliser votre OS librement
